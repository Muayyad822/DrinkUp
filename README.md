## Getting Started ğŸš€

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/muayyad822/drinkup.git
cd drinkup
```

2. Install dependencies
```bash
npm install
# or

```

3. Start the development server

```bash

```
